# pyqt6-forms

How to install:
<p> 
```
    pip install pyqt6_forms
    pip3 install pyqt6_forms
```
</p>

How to use:
<p> 
```python
    from pyqt6_forms.forms import QLineEditForm
    ....
    form = QLineEditForm(form_name='username', parent=QWidget |QHBoxLayout)
    ...
    user_input_username = form.get_value()
```
</p>
